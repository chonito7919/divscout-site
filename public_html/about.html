About DivScout

Building a clean dividend payment database from SEC filings

Mission

DivScout aims to provide investors and researchers with reliable, validated dividend payment data extracted directly from official SEC EDGAR filings. By processing XBRL data and applying intelligent quality checks, we create a clean database of historical dividend information.

Data Quality

Our confidence scoring system evaluates each dividend record based on multiple factors:

  • Detection and filtering of annual totals vs. individual payments
  • Validation of payment dates and amounts
  • Identification of potential data quality issues

Open Source

DivScout is built as an open-source project with two main components:

  • Backend Parser: github.com/chonito7919/DivScout
  • Web Dashboard: github.com/chonito7919/divscout-web

Current Status

DivScout is currently in active development with:

  • Coverage of select companies (AAPL, JNJ, TGT, and expanding)
  • Historical dividend data from SEC filings
  • Batch-updated data processing
  • Continuous improvements to validation algorithms
